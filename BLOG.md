# Automate dYdX Perpetual Trading with Our New n8n Community Node

We're excited to announce that Velocity BPA has released a powerful new community node for n8n: the dYdX DEX integration. This node brings decentralized perpetual trading automation directly into your n8n workflows.

## The Challenge of DeFi Trading Automation

Traders and DeFi protocols often struggle with automating their dYdX operations. Whether it's monitoring positions, executing trades based on market conditions, or participating in governance, manual processes create delays and missed opportunities. Traditional automation tools lack native blockchain integrations, forcing developers to write custom code or manage complex API connections.

## Introducing n8n-nodes-dydx-dex

Our new community node eliminates these friction points by providing a no-code interface to dYdX's powerful perpetual trading platform. Now you can build sophisticated trading workflows, governance automation, and real-time monitoring systems without writing a single line of code.

## Key Features

**Perpetual Trading Operations**: Execute market and limit orders, manage positions, and query order books directly from your workflows. Perfect for implementing automated trading strategies.

**Staking Management**: Automate your staking operations, monitor rewards, and optimize your yield strategies with scheduled workflows.

**Governance Participation**: Stay engaged with dYdX governance by automating proposal monitoring and voting actions.

**WebSocket Triggers**: React to market events in real-time with WebSocket support. Build workflows that trigger on price movements, order fills, or position updates.

## Getting Started

Installing the node is simple. In your n8n instance, run:


npm install n8n-nodes-dydx-dex


After restarting n8n, you'll find the dYdX node in your node palette, ready to integrate with your existing workflows.
